# Análisis Geoespacial y Estadístico con Python

Este repositorio contiene una guía técnica interactiva desarrollada en Jupyter Notebook para el procesamiento de datos raster multiespectrales y el análisis estadístico descriptivo.

## Contenido del Proyecto

* **Procesamiento Raster (Sentinel-2):** Manipulación de bandas satelitales utilizando `rasterio`, generación de composiciones en color real e infrarrojo, y cálculo automatizado de los índices ambientales **NDVI** (Vegetación) y **NDWI** (Agua).
* **Análisis Estadístico Comparativo:** Evaluación del rendimiento y consistencia temporal entre sensores térmicos de bajo costo (Arduino) y sensores de estándar comercial (Hobo) mediante representaciones gráficas avanzadas (`matplotlib` y `seaborn`).

## Tecnologías Utilizadas
* Python 3
* Rasterio & NumPy (Procesamiento numérico y geoespacial)
* Pandas (Gestión de bases de datos)
* Matplotlib & Seaborn (Visualización estadística de datos)
